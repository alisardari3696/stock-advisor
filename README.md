# STOCK ADVISOR DEMO

A Python-based demo tool for visualizing cumulative growth of assets in Iran's market, including inflation, USD price, equal-weight index, and the stock "فملی".

## Features
- Auto-fetches and caches financial data
- Log-scale chart for visual comparison
- Persian language support
- Simple GUI for selecting date range

## How to Run
Run `stock_advisor.py` with Python 3.9+ and required libraries listed in `requirements.txt`.
